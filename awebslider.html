Slides structured by HTML5.

A Web-based slideshow is a presentation that can be viewed or presented using a web browser.1

AWebSlider presentation is contained inside an article html tag and a slide is a section of it.

  • No need for a carousel.
  • Works without JavaScript.

The slide

<section id="this_slide_id">
<h1>Header for slide.</h1>
Your Content Here!
<nav>
<a href="#id_of_previous"></a>
<a href="#this_slide_id" class="s"></a>
<a href="#id_of_next"></a>
</nav>
</section>
  • The slide's id can be any valid CSS3 identifier you want.
  • No need to add the slide's id to the stylesheet.
  • Only one nav tag can be placed anywhere within the section tag.

CSS3 enables the user interface.

AWebSlider uses CSS3 to format the appearance of the presentation.

:not(:target) :target {z-index}

The :target selector and the z-index property are the magic that makes it possible.

section						{ z-index: 2; }
section:not(:target)				{ display: none; }
section:target					{ display: block; z-index: 4; }
section:first-of-type ~ section:last-of-type	{ display: block; z-index: 1; }
section:target ~ section:last-of-type		{ display: block; z-index: 3; }
section:first-of-type				{ display: block; }

Compass Outlines the Table of Contents.

The same content is used to generate slide thumbnails.

[checkbox]:checked

This is it! Tons of javascript not needed! Almost!PS

article > [type=checkbox]:checked ~ section

JavaScript is used although it is not required.

A click event is used to uncheck the checkbox that activated the Compass menu.

Still usable if JavaScript is disabled.

  1. While the Compass menu is active.
  2. Click the slide you want to show.
  3. Click the close button X (top right) of the Compass menu.
  4. Slide loads!
